# bug-free-prog

Simple collection of useful Python scripts for random generation.

### Scripts

- **`random_code.py`** — Generates random 6-digit numeric codes (great for OTP, verification, testing)
- **`random_number.py`** — Returns a random integer from 1 to 100
- **`random_password.py`** — Strong random password generator with customizable length and special characters
- **`random_color.py`** — Generates random HEX colors (useful for design and testing)

### How to use

```bash
python random_code.py
python random_password.py
python random_color.py
